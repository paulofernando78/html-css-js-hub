Button Show / Hide

Hey there!

HTML


<button id"toggle-button">Click me!</button>
<div id"element">Hey there!!</div>

CSS


#element {
  display: none;
  background: red;
  padding: 10px;
  width: fit-content;
}

JS


main.js
import { showHide } from "./lessons/show-hide.js"
                
document.getElementById("toggle-button").addEventListener("click", () => {
showHide("element")
})

button-show-hide.js
export function showHide(id) {
  const element = document.getElementById(id);
  if (element.style.display === "block") {
    element.style.display = "none";
  } else {
    element.style.display = "block";
  }
}

Notes

You don't need the onclick attribute in the HTML because you are already adding an event listener in your JavaScript. The addEventListener method is a better practice since it separates HTML from JavaScript logic, keeping your code cleaner and more maintainable.

Using an event listener is generally preferred because it provides better separation of concerns. It's also more flexible, allowing you to attach multiple listeners to the same element, whereas onclick replaces any existing inline event handler.
